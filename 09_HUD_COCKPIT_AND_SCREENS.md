# Prompt 09 — HUD, cockpit framing, and screens

Build presentation quality.

Implement:
- title screen / attract-like presentation
- HUD
- score display
- lives/health/energy display as appropriate
- wave/status messaging
- game over screen
- restart flow

Optional if helpful:
- cockpit frame accents
- target box indicators
- horizon/grid/reference lines
- intro transition into play

Requirements:
- HUD must be readable and elegant
- do not clutter the screen
- the game should look intentional in still screenshots

Verify:
- title screen looks like a real game, not a placeholder
- in-game composition is clear and attractive
