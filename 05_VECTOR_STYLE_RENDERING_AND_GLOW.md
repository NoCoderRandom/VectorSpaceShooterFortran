# Prompt 05 — Vector style rendering, glow, and presentation

Make the renderer start looking impressive.

Improve the renderer so it evokes classic color vector arcade style:
- black background
- bright colored lines
- slight glow or persistence illusion
- line brightness or thickness variation where useful
- optional multi-pass line draw trick if appropriate

Add:
- sparse starfield with depth feel
- restrained bloom-like feel using simple redraw/persistence if feasible
- readable central crosshair/reticle

Do not chase photorealism.
Do chase clean arcade spectacle.

Verify:
- screenshot-worthy frame
- clear visual improvement over raw single-pass lines
- still readable and performant
