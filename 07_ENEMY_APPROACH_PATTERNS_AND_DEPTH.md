# Prompt 07 — Enemy approach patterns and depth

Implement enemies that make the perspective system meaningful.

Requirements:
- enemies approach from depth toward the player
- scaling/projection/motion create real tension
- support several simple movement patterns
- behavior is arcade-readable, not random noise

Add:
- spawn manager
- formation or wave logic
- at least 3 motion archetypes

Examples:
- straight-in attack
- weaving approach
- lateral sweep across the aim point

Verify:
- enemies read clearly at distance
- approaching enemies become more threatening visually
- gameplay space starts feeling alive
