# Prompt 08 — Hits, damage, and feedback

This prompt is about game feel.

Implement:
- hit detection suitable for vector targets
- enemy destruction feedback
- player hit feedback
- score feedback
- game over feedback
- brief but satisfying explosion or breakup effect using vector-style lines/particles

Requirements:
- every successful hit is obvious
- every player mistake is obvious
- feedback is readable at arcade speed
- visual language stays consistent

Add:
- impact flashes
- vector debris, spark lines, or breakup
- short screen shake or camera kick if helpful

Verify:
- combat feels much more satisfying after this pass
