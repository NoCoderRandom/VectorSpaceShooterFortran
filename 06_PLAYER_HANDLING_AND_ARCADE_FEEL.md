# Prompt 06 — Player handling and arcade feel

Implement player control with strong arcade feel.

Requirements:
- responsive input
- smooth movement or aiming suitable for the chosen perspective
- clear fire control
- no sluggishness
- no simulation-heavy drift unless it improves arcade feel

Decide and document the control model:
- crosshair aim + forward motion
or
- ship-centered aim with enemy approach

Add:
- shot cadence that feels good
- immediate visible firing feedback
- basic recoil/flash/reticle response if appropriate

Verify:
- controls feel responsive
- firing feels intentional and readable
