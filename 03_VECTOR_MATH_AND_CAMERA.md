# Prompt 03 — Vector math and camera foundation

Implement the 3D math core.

Include:
- vec2/vec3 style support
- transform representation
- rotation helpers
- translation helpers
- perspective projection
- camera/view transform
- near-plane safety checks

Keep it practical and readable.

Create a visual test scene:
- one rotating wireframe cube or ship-like frame
- perspective projection clearly visible

Verify:
- apparent size changes with depth
- rotation is stable
- no projection explosions
